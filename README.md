# ClipBox Chrome 扩展

一个高效管理常用文本片段的Chrome扩展，让你快速访问、复制和管理各类常用短语、指令和文本内容。可以用来存临时账号密码token或AI指令。本地缓存。支持导入导出。

## 功能特性

- 📝 **文本片段管理**：轻松添加、编辑、删除各类常用文本内容
- 🏷️ **分类标签**：为文本片段添加分类，便于快速查找
- 🔍 **智能搜索**：支持按内容和分类搜索文本片段
- 📋 **一键复制**：点击即可将内容复制到剪贴板
- 📥 **剪贴板粘贴**：快速从剪贴板添加新的文本片段
- 📤 **导出/导入**：支持JSON格式数据导出和导入，便于备份和迁移
- ⚙️ **个性化设置**：自定义扩展标题和图标
- 💾 **本地存储**：所有数据存储在本地，保障隐私安全
- 🎨 **美观界面**：现代化的UI设计和流畅动画
- 📱 **响应式设计**：适配不同屏幕尺寸
- 🚀 **快速操作**：直观的用户界面，提供高效的操作体验

## 安装方法

1. **下载文件**：将所有扩展文件下载到本地文件夹
2. **打开Chrome扩展页面**：在Chrome地址栏输入 `chrome://extensions/`
3. **启用开发者模式**：点击右上角的"开发者模式"开关
4. **加载扩展**：点击"加载已解压的扩展程序"，选择包含扩展文件的文件夹
5. **完成安装**：扩展图标将出现在Chrome工具栏中

## 使用指南

### 启动扩展
- 点击Chrome工具栏中的ClipBox扩展图标
- 扩展面板将弹出显示

### 基础操作
1. **新增片段**：点击"新增"按钮，输入内容和分类，保存即可
2. **粘贴内容**：点击"粘贴"按钮，自动从剪贴板获取内容并添加（需授权）
3. **复制内容**：点击任意文本片段上的"复制"按钮，内容将自动复制到剪贴板
4. **编辑片段**：点击"编辑"按钮修改现有内容和分类
5. **删除片段**：点击"删除"按钮移除不需要的文本片段

### 搜索和筛选
- 在搜索框中输入关键词查找内容
- 使用分类筛选器按类别查看文本片段

### 导入导出
- **导出**：点击"导出"按钮，将所有内容保存为JSON文件
- **导入**：点击"导入"按钮，选择之前导出的JSON文件，可以选择覆盖现有数据或追加

### 个性化设置
- 点击顶部的"设置"按钮
- 可以自定义扩展标题和图标样式
- 支持的心形、猫咪、书籍、机器人和盒子等多种图标选择

## 技术特性

- **Manifest V3**：使用最新的Chrome扩展规范
- **本地存储**：利用Chrome Storage API保存用户数据
- **剪贴板API**：支持读取和写入剪贴板内容
- **响应式设计**：适配不同屏幕尺寸
- **模态对话框**：使用原生HTML Dialog实现弹窗交互
- **事件委托**：高效处理列表项点击事件
- **UUID生成**：使用浏览器原生API生成唯一标识符

## 注意事项

1. **权限需求**：扩展需要访问本地存储和剪贴板的权限
2. **隐私保护**：所有数据仅存储在本地，不会上传到任何服务器
3. **剪贴板访问**：首次使用粘贴功能时需要授权
4. **数据安全**：建议定期导出数据进行备份，卸载扩展会删除所有存储的数据
5. **浏览器兼容性**：仅支持Manifest V3兼容的Chrome浏览器版本

## 常见问题

**Q: 数据存储在哪里？**
A: 所有数据都存储在浏览器的本地存储中，完全私密，不会上传到云端。

**Q: 如何备份我的数据？**
A: 使用"导出"功能将数据保存为JSON文件，需要时可通过"导入"功能恢复。

**Q: 粘贴功能为什么需要权限？**
A: 根据Chrome的安全策略，读取剪贴板内容需要用户明确授权。

**Q: 如何恢复默认设置？**
A: 可以在设置面板中将标题和图标重置为默认值。

**Q: 导入数据时是否可以选择合并而不是覆盖？**
A: 是的，导入时会提示是否覆盖现有数据，选择"取消"则会合并追加。
